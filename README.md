# Gemini Desktop

WebアプリのGeminiをTauriでラップしだだけ

<img width="300" alt="スクリーンショット 2026-04-16 214151" src="https://github.com/user-attachments/assets/9b13e19b-c4dc-439f-ac07-3493a4eb268f" />

## Chat内容を指定して起動

~~~bash
C:\Program Files\Gemini Desktop\tauri-app.exe "こんにちは"
~~~

## PowerTypes Command Pallet組み合わせ

Command Pallet組込のブックマーク機能の起動パスを以下のようにしておけば

~~~bash
C:\Program Files\Gemini Desktop\tauri-app.exe "{Query}"
~~~

Command PalletからChat内容指定して起動可能
